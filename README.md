# Fundamentos de IA e Análise de Dados - SENAI

Este repositório armazena o material didático, notas e exercícios desenvolvidos durante o programa **FIC (Formação Inicial e Continuada)** de **Fundamentos de IA e Análise de Dados** do **SENAI**. O objetivo do espaço é organizar o progresso acadêmico e as implementações práticas realizadas nas aulas.

## 📂 Estrutura do Repositório

O projeto conta com uma organização que abrange desde fundamentos até aplicações de modelos avançados:

*   **Análise de Dados e ML:** `analise_dados.ipynb` e `machine_learning.ipynb`.
*   **Visão Computacional:** `visao.py` e o modelo `yolov8n.pt`.
*   **Aulas e Exercícios:** Notebooks numerados de 01 a 05 e exercícios práticos.

## 🛠️ Tecnologias e Bibliotecas

Para o desenvolvimento das atividades, foram utilizadas as principais bibliotecas do ecossistema de Ciência de Dados e IA do Python:

### **Análise e Manipulação de Dados**
*   **Pandas:** Utilizada para a manipulação e análise de dados estruturados (tabelas), essencial nos notebooks de análise de dados e relatórios.
*   **NumPy:** Suporte para arrays e matrizes multidimensionais, além de funções matemáticas de alto nível.

### **Visualização de Dados**
*   **Matplotlib & Seaborn:** Empregadas na criação de gráficos e visualizações estatísticas, facilitando a extração de insights presentes em `aula_05_relatorios.ipynb`.

### **Inteligência Artificial e Machine Learning**
*   **Scikit-learn:** Principal ferramenta para implementação de algoritmos de aprendizado de máquina (como regressão e classificação) em `machine_learning.ipynb`.
*   **Ultralytics (YOLOv8):** Utilizada especificamente para tarefas de **Visão Computacional**, como detecção de objetos, conforme indicado pelo arquivo `yolov8n.pt` e o script `visao.py`.

### **Ambiente de Desenvolvimento**
*   **Jupyter Notebook:** O repositório é composto quase inteiramente por notebooks interativos (.ipynb), que permitem a união de código, texto e visualizações.

## 🚀 Como Executar

1.  Certifique-se de ter o Python instalado.
2.  Instale todas as dependências listadas acima (ou use o arquivo disponibilizado):
    ```bash
    pip install -r requirements.txt
    ```
3.  Abra os notebooks utilizando o **Jupyter Lab** ou **VS Code**.
